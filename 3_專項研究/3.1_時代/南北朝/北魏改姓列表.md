# 北魏改姓列表
- 部分摘自《魏書》卷113官氏志第十九：自古天子立德，因生以賜姓……
    - 十姓：魏孝文帝弟兄、親屬等，互不通婚
    - 統帥諸部：起自拓拔力微時諸姓，原先皆為某部統帥，尉遲以下混得還不如四方諸部
    - 四方諸部：歲時朝貢的四方諸部
    - 補遺：通鑑中有記載，不見於魏書，可能包括未存活至魏孝文帝時期的姓氏
- 魏孝文帝改姓
    - 漢人四姓：盧崔鄭王，后加李為五姓，分別爲范陽盧氏、清河崔氏、滎陽鄭氏、太原王氏、趙郡李氏
    - 勛臣八姓：穆陸賀劉樓于嵇尉
