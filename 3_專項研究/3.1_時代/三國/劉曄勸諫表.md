# 劉曄勸諫表
- 劉曄少年精於武，雷厲風行，快意恩仇，有俠客之風。中年精於文，預言從未出錯，而幾乎從未被採納。晚年精於官，迎合上意，實為無奈不爭之舉，不料竟因此獲罪而死。
